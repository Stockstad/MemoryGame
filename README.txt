############# UI:s 
startform -> Starting UI with menu.
settingsform -> UI for settings and settings selections.
playerControl -> UI for character selection.
Form (Backup) -> Main UI. Handles the game mechanics. 
FinalForm -> Final UI for results display.

############# Classes
Game -> Static class; GameManager.
Playable -> Playable Characters.
Player -> Player information. Score, movesmade etc.
PlayingCard -> Cards's information. 
Sounds -> Static class for handling of sounds and music.

############ Rescources
Pictures -> Playing cards's pictures.
players -> Players's pictures.
players_blur -> Blurry pictures for the players.
rest -> Other recourses. 
Sound -> Sound and music rescourses.
hints -> (As of version 1.2, not in use).
